# cm-classifiers.py
Retrieve through snmp the service flows and classifiers of a cable modem and print them.

Usage:

```sh
$ python cm-classifiers.py ip snmp-read-community
```

### Example
```sh
$ python cm-classifiers.py 192.168.0.1 public
```
